# Análisis Exploratorio de Ventas (EDA)

Este proyecto presenta un análisis exploratorio de un conjunto de datos de ventas, realizado en un notebook de Jupyter utilizando herramientas básicas de Python.

# Objetivo
Explorar y comprender el comportamiento de las ventas mediante estadísticas descriptivas y visualizaciones, con el fin de identificar patrones, tendencias y posibles anomalías en los datos.

## Herramientas utilizadas
Python
Pandas
Matplotlib

## El notebook incluye:
Carga de datos
Exploración inicial del dataset
Limpieza básica de datos
Análisis descriptivo
Visualización de datos con gráficos
Resultados

A través del análisis se pueden observar:
- Tendencias generales en las ventas
- Distribución de variables clave
- Posibles valores atípicos o comportamientos inusuales
